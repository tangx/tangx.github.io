本文所有代码就基于 windows 版本的python 2.7.11 x86

1. 最基本抓站

对于抓取的对象, 都会使用正则方式进行匹配。

编写正则的一个小技巧：

1.将整个resp.read()文本复制下来, 使用删除法进行匹配, 删哪里匹配哪里。

  1. 注意通配符 + | * 的的贪婪模式， 如果不需要贪婪模式，可以使用 ? 打断。 +? | *?

  2. 如果要在多行之间进行匹配, 需要使用标记符号 patt=re.compile( str, flags=re.MULTILINE + re.DOTALL)

2. 使用代理服务器

2.1 第一种代理方式

在windows下时候socket代理的时候, 出现了 socket.inet_pton 错误.
错误原因是 windows下的socket.py 模块没有 inet_pton 属性.
解决方法当然就是 安装 win_inet_pton , 并在 socket.py 中 import.
具体操作, 参考 Windows下socket inet_pton 错误

3.4 反"反盗链"

一些网站反盗链的机制, 说穿了就是查看请求是否来自己的网站。 因此在headers信息中添加Referer信息即可。



headers={
'Referer':'http://www.cnbeta.com/articles',
}
