2. 字段类型

  1. char 和 varchar 是不同的。 前者会补全所有长度， 后者不会。

    • 坑: passwd 为 char(255)， 导致 select 时导致屏幕没有 token 显示， 认为反弹失败。

  2. 根据 syscolumns 中的列信息，构建接受表。

  3. syscolumns表内的xtype
